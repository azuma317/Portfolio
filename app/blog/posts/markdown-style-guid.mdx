# Markdown スタイルガイド

## はじめに

このドキュメントは、様々な Markdown 要素のスタイリングを確認するためのサンプルです。
**太字**、_イタリック_、そして **_太字かつイタリック_** などの基本的なテキストスタイリングを含みます。

## テキストの装飾

- ~~打ち消し線~~ を使用した文字
- `インラインコード` の表示
- [リンクテキスト](https://example.com)
- ==ハイライト== テキスト

## リスト表示

### 番号付きリスト

1. 最初のアイテム
   1. ネストされたアイテム 1
   2. ネストされたアイテム 2
2. 二番目のアイテム
3. 三番目のアイテム

### 箇条書きリスト

- メインアイテム 1
  - サブアイテム 1
  - サブアイテム 2
- メインアイテム 2
  - サブアイテム 1
    - さらにネストされたアイテム

## 引用ブロック

> これは引用ブロックです。
>
> 複数行の引用も可能です。
>
> > ネストされた引用ブロック
> >
> > > さらにネストされた引用
